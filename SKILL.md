# Javelina DNS Management Platform

A modern DNS management platform built with Next.js, offering enterprise-grade DNS record management, multi-organization support, and comprehensive billing integration.

## Description

Javelina is a full-stack DNS management application that enables users to manage DNS zones and records across multiple organizations. The platform features:

- DNS zone and record management with real-time updates
- Multi-organization support with role-based access control
- Stripe integration for subscription billing and plan management
- Admin panel for user, organization, and system management
- Audit logging for all administrative actions
- hCaptcha integration for bot protection on authentication flows
- Dark mode support with theme switching
- Responsive design for desktop and mobile devices

Built with Next.js 15, TypeScript, Supabase for authentication and database, Tailwind CSS for styling, and integrates with an Express.js backend API for DNS operations.
